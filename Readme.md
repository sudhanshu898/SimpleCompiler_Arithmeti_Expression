Web UI
------

Open `index.html` in a browser to use the new UI.

Commands:

```powershell
# (Windows) build the C++ version
g++ -std=c++11 -o SimpleCompiler main.cpp ExpressionTree.cpp
.
\SimpleCompiler

# Or open the web UI in a browser (no build needed):
start index.html
```
